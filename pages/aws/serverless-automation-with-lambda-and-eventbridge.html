Serverless automation with Lambda and EventBridge

June 22, 2021

Idea

  • Automation can be done easily and cheaply.
  • Serverless means you only need to upload a program and set a schedule, and AWS takes care of maintaining the system running your program.
  • Lambda is for you to upload program that you want to run, and EventBridge is to set schedule running the program

Lambda

  • Upload Python program and Python packages together in a zip file

EventBridge

  • EventBridge can trigger AWS Lambda by rule type: schedule expression in cron style.
  • Go to EventBridge, Create rule
  • Define pattern
    • Schedule
  • Select event bus
    • AWS default event bus
  • Select targets
    • Target: Lambda function
    • Function: Select your function

Cost

  • Lambda
    • How many times you run a function
    • How long the function takes
    • How much memory is allocated to the function
  • EventBridge
    • It’s free for EventBridge to trigger Lambda
  • Example
    • Automating a Lambda function which runs daily, and spends 1 minute and 1GB memory is $0.03 per month.

Cron expression

  • The cron expression of AWS EventBridge is a bit different from Linux cron job expression.
  • e.g. every 5 minutes
    • EventBridge: cron(0/5 * * * ? *)
    • Linux cron job: */5 * * * *
