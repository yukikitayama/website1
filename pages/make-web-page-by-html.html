Make web pages by HTML

Structure

  • Build the following structure in <body>
    • <header>
    • <nav>
      • Navigational links to other pages within the same website, table of contents, could be in <header> and <footer>
    • <article>
      • For blog posts
    • <section>
      • Used to make hierarchy to a page
    • <aside>
      • Sidebars
    • <footer>

Markup validation

  • Use https://validator.w3.org/ to check your html file is fine.
  • Because errors propagate, work on fixing at the top.
  • Even professional websites show error in this validator when you enter their URI.

Accessibility validation by code

  • Go to https://wave.webaim.org/
  • Enter your website URL and turn off styles

Visual accessibility validation

  • Go to https://www.funkify.org/
  • It simulates the visual appearance from the users with different disabilities and difficult situations.

Input pattern

  • A source of regularly used input pattern.
  • Go to https://www.html5pattern.com/
