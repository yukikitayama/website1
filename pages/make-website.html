Make a website

Overview

  • Make a S3 bucket
  • Make it public
  • Use Bucket website endpoint to see the website

Bucket structure

  • Put index.html as your home page
  • If you create a folder structure in your bucket, you must have an index document at each level.

Error document

  • By default, Amazon S3 returns a predefined HTML message, 403 Forbidden response.

Website access permission

  • Turn off Block all public access
  • Add a bucket policy

Server access logging

  • Give you records for the requests that are made to your bucket.
  • Amazon S3 writes website access logs to your log bucket every 2 hours.

CloudFront to speed up website

  • https://docs.aws.amazon.com/AmazonS3/latest/userguide/website-hosting-cloudfront-walkthrough.html

Test website

  • Use Bucket website endpoint, such as, which you can find in Static website hosting in Properties in the S3 bucket.
