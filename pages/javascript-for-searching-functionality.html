JavaScript for searching functionality

Idea

  • Suppose many items are listed by <li>, and you want to see only the list items which contain a certain keyword.
  • Make an input box by <input>. When a user types a keyword in the input box, oninput Event executes a JavaScript function which turns on or off the list display.

HTML

<body>

<input type="text" id="searchInput" oninput="controlListDisplay()">

<ul>

    <li>apple</li>

    <li>orange</li>

    <li>banana</li>

</ul>

</body>

JavaScript

function controlListDisplay() {

    var word = document.getElementById('searchInput').value;

    word = word.toLowerCase();

    var lis = document.getElementsByTagName('LI');

    for (i = 0; i < lis.length; i++) {

        if (!lis[i].innerHTML.toLowerCase().includes(word)) {

            lis[i].style.display = 'none';

        }

        else {

            lis[i].style.display = 'list-item';

        }

    }

}
