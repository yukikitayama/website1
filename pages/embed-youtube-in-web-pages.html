Embed YouTube in web pages

Steps

  • Get URL of a YouTube video
  • Use <iframe> with src=”YouTube URL”
  • You need to modify YouTube URL as following if it doesn’t display
    • https://www.youtube.com/embed/ + Unique YouTube ID usually an ending part.
    • If your URL is like www.youtube.com/watch?v= + Unique YouTube ID or youtu.be/ + Unique YouTube ID, probably you need to modify.

Style

HTML

<section class="youtubeContainer">

    <iframe src="https://www.youtube.com/embed/9fmXC3lyDCw"></iframe>

</section>

CSS

.youtubeContainer {

    position: relative;

    overflow: hidden;

    width: 100%;

    padding-top: 56.25%;

}

section iframe {

    border: none;

    /*margin: 0 auto;*/

    /*display: block;*/

    position: absolute;

    top: 0;

    left: 0;

    bottom: 0;

    right: 0;

    height: 80%;

    width: 80%;

}
