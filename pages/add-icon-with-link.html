Add icon with link

Font Awesome

  • Popular and easiest way to use icons. Many major icons are free.
  • To start using Font Awesome, visit https://fontawesome.com/start and register your email.

Example

Add Linkedin icon with link to your Linkedin profile

  • Add Font Awesome kit’s code to your <head>
    • <head>
    •     <!-- Font Awesome kit's code -->
    •     <script src="https://kit.fontawesome.com/your_kits_code.js" crossorigin="anonymous"></script>
    • </head>
  • Add Linkedin icon with link
    • <a href="your_linkedin_profile_url"><i class="fab fa-linkedin" style="color: #0a66c2;"></i></a>

Add YouTube icon with link to your YouTube channel

  • Add the same Font Awesome kit’s code to head
  • Add YouTube icon with link
    • <a href="your_youtube_channel_url"><i class="fab fa-youtube" style="color: #ff0000;"></i></a>
