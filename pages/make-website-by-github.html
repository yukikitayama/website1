Make a website by GitHub

Idea

  • To publish a website, you need a domain name and hosting service. By using GitHub, your repository name will be the domain name and the repository also will be your hosting service. 
  • It’s free.

Steps

  • Make a repository
  • Upload everything
  • Go to settings of the repo
    • Go to Pages section
    • Select Branch (Usually main) and save
    • It will give you URL to your website
      • If the URL does not show it, add /index.html to the URL.
