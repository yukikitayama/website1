Type hints in Python

Idea

  • Type hints add type information of function arguments and return value.
  • argument use “: type” after arguments, and return value use “-> type” after end parenthesis and before colon.
  • We can make a clean and documented code
  • Introduced in Python 3.5

Example

  • Integer type hint.

def func1(n: int) -> int:

    return n * n

func1(2)

4

  • Even if you use something other than int and as long as Python can run, it returns and does not throw an error.

func1(2.1)

4.41

  • If you use types other than built-in types, such as a list, you need to import the typing module.

from typing import List

def func2(nums: List[int]) -> int:

    return max(nums)

func2([1, 2, 3])

3
