Static inner class for the main method in Java

Issue

  • When you try to make an object of an inner class in the main method, you receive the error “non-static variable this cannot be referenced from a static context”.

Example

public class Solution {

    public class SomeClass {

        public int someInteger;

    }

    public static void main(String[] args) {

        SomeClass test = new SomeClass();

    }

}

Error

  • java: non-static variable this cannot be referenced from a static context

Solution

  • Make the inner class static by public static class SomeClass.
    • An inner class contains an implicit reference to an instance of the parent class. It means that you need to make an instance of the outer class Solution in the main method.
    • But we want to use SomeClass without making a Solution instance, so we make the inner class static in order to be able to use it without an instance.

public class Solution {

    public static class SomeClass {

        public int someInteger;

    }

    public static void main(String[] args) {

        SomeClass test = new SomeClass();

    }

}
