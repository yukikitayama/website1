Style web pages by CSS

Order of style applied (Cascading)

  1. Browser default
  2. External style sheets
  3. Internal style (<style> in <head>)
  4. Inline style (e.g. <h1 style=”something” inside HTML elements)

Font

  • Use font-family
  • Use several font names to let browsers try the next front if it’s not supported. These are separated by commas.
    • e.g. font-family: Calibri, Sans-serif;
    • Good to start with the font you want and end with a generic family.
