center() method useful for debugging

Idea

  • You want to print as many strings or variables as possible, because you want to debug or to check whether your program is running fine.
  • But you think it’s annoying to type a sequence of ‘*****’ or ‘-----’ to make blocks in standard output, because the length changes.
  • Use center() to fix the length of the string and ‘***’ and make a concise code.

Example

  • center() is a method of Python String.
    • The first argument sets the output string length.
    • The second argument sets a string which fills in the rest of the string by the length minus the Python string.

print(' Test '.center(30, '*'))

************ Test ************

long = 'This is a long text'

print(f' {long} '.center(30, '*'))

**** This is a long text *****

short = 'Short'

print(f' {short} '.center(30, '*'))

*********** Short ************

  • The output is always the same length if you fix the first argument, and you don’t have to repeat the same string in typing.
